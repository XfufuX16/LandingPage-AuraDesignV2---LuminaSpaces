# Lumina Spaces - Art of Living

Landing page for **Lumina Spaces**, an architectural and interior design studio. Built with a clean, editorial aesthetic that emphasizes elegance and minimalism.

## Preview

A modern, responsive landing page featuring:

- **Hero Section** — Bold typographic header with portfolio imagery
- **Quote Section** — Inspirational architecture quote
- **Featured Stories** — Journal-style editorial articles with hover effects
- **Services** — Residential, Commercial, and Landscape design categories
- **Footer** — Newsletter signup, sitemap, and contact information

## Tech Stack

- **HTML5** — Semantic markup with accessibility best practices
- **Tailwind CSS** (via CDN) — Utility-first styling
- **Google Fonts** — Playfair Display (serif) & Mulish (sans-serif)
- **Material Symbols** — Icon set

## Design Features

- Warm, paper-like background texture
- Muted gold accent color (`#C5A059`)
- Editorial vertical side navigation (desktop)
- Smooth image zoom on hover
- Fully responsive layout (mobile + desktop)
- Custom scrollbar styling

## Getting Started

Simply open `index.html` in a browser — no build step required.

```bash
# Clone the repository
git clone https://github.com/XfufuX16/LandingPage-AuraDesignStudioV2.git

# Open in browser
start index.html
```
